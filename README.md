## 说明
此存储库是为了攀枝花学院数学与计算机学院学生准备期末考试所建立的，作为学院教务系统课程资源的备份。<br>
复习题文档采用md格式或是pdf格式，而不是Word格式。<br>
在对应目录上传时，需要说明文档的，复习题题型，年份，有无缺失，有无答案及其他能够说明此文档大概内容的命名。<br>
文件格式如：思修_2019-6_复习重点_无缺失.md
